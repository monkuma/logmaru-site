# LogMaru Website

Official website and download page for **LogMaru**.

LogMaru is a local-first Windows desktop application for categorized logging and time tracking.  
It helps you record activities, organize them by category, and visualize your time usage — all stored locally.

This repository contains the source code for the public website.  
The desktop application is maintained separately.

---

## Tech Stack

### Website
- Astro
- Cloudflare Pages
- Markdown-based content management

### Desktop Application
- Rust
- Tauri2
- SQLite
- React

---

## Development

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

---

## Deployment

The site is deployed automatically via Cloudflare Pages.

Pushing to the `main` branch triggers a production build.

This website is planned to support both English and Japanese.
